# 病院勤務を続けながら、トレーナー活動を始める方法 — PT向け無料オンラインセミナー LP

さくら通り整形外科クリニック主催、理学療法士（PT）向け無料オンラインセミナーのランディングページ。

- 公開ページ: `index.html`（依存なしの静的 HTML 1ファイル / ビルド不要）
- 対象: 経験年数3〜10年目の理学療法士
- 開催形式: オンライン

## 申込ボタンのリンク先

すべての「無料でセミナーに申し込む」ボタンは仮リンク `#apply` です。
Google フォームの URL が決まったら一括置換してください:

```bash
sed -i 's|href="#apply"|href="<FORM_URL>"|g' index.html
```

## 写真の差し込み

ヒーロー／院内・リハビリ室／講師／面談 の4箇所はプレースホルダー（点線枠）です。
実写真が用意できたら該当の `<div role="img" …>` を `<img>` に置き換えてください。
